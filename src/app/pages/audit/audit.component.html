Audit "Audita" ou "filtra" os valores enviados.

1. O Audit filtra os valores enviados, mantendo apenas os últimos n valores enviados.

2. O Audit é um pouco mais complexo que o ReplaySubject, pois ele grava os valores enviados e os envia para os observadores.

Obs.: Abra o console para ver o resultado.

Obs2.: Abra o arquivo audit.component.ts e veja o código fonte.
