BufferCount Armazena os valores observáveis de origem até que o tamanho atinja o bufferSize máximo fornecido.

1. Coleta valores do passado como um array e emite esse array somente quando seu tamanho atinge bufferSize.

2. Parametro: bufferSize - O tamanho máximo do buffer emitido.

3. Parametro: startBufferEvery (opcional. Valor padrão é null) - Intervalo no qual iniciar um novo buffer. Por exemplo, se startBufferEvery for 2, um novo buffer será iniciado em todos os outros valores da origem. Um novo buffer é iniciado no começo da origem por padrão.

Obs.: Abra o console para ver o resultado.

Obs2.: Abra o arquivo buffer-count.component.ts e veja o código fonte.
