BufferToggle Ative para capturar valores emitidos da fonte, desative para emitir valores em buffer como array.

1.O BufferToggle armazena os valores enviados e emite uma lista de valores quando o número de valores é atingido.

2. Parametro: openings - Um Subscrible ou Promise de notificações para iniciar novos buffers.

3. Parametro: closingSelector - Uma função que pega o valor emitido pelos observables abertos e retorna um subscrible ou Promise, que, ao emitir, sinaliza que o buffer associado deve ser emitido e limpo.

Obs.: Abra o console para ver o resultado.

Obs2.: Abra o arquivo buffer-toggle.component.ts e veja o código fonte.
