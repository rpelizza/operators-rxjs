Buffer Armazena os valores observáveis de origem até que o closureNotifier seja emitido.

1. O Buffer armazena os valores enviados e emite uma lista de valores quando o número de valores é atingido.

2. O Buffer não envia valores que não estão dentro do intervalo de tempo especificado.

3. Um comportamento interessante do operador de buffer é que ele pode enviar um conjunto vazio de valores para o observador se o notificador observável emitir um valor e o cache estiver vazio.

Obs.: Abra o console para ver o resultado.
