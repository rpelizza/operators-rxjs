Observable Objeto que pode emitir eventos

Um Observable é um objeto que pode emitir eventos.

Clique aqui para ver o exemplo

Subscription Objeto que armazena uma assinatura de um Observable

O objeto Subscription armazena uma assinatura de um Observable. Quando o Observable é finalizado, a assinatura é cancelada.

Clique aqui para ver o exemplo

Subject Um Subject é um tipo especial de Observable que permite que valores sejam disparados para muitos Observadores.

Enquanto os Observables simples são unicast (cada Observer inscrito possui uma execução independente do Observable), os Subjects são multicast. Um Subject é como um Observável, mas pode transmitir para muitos Observadores.

Clique aqui para ver o exemplo

Behavior Subject Um Behavior Subject é um Subject que inicializa seu valor com o último valor enviado.

Requer um valor inicial e emite o valor atual para novos subscribers.

Clique aqui para ver o exemplo

Replay Subject "Reproduz" ou emite valores antigos para novos subscribers.

ReplaySubject é semelhante ao BehaviorSubject na maneira como pode enviar valores em cache para novos assinantes, mas em vez de apenas um valor atual, ele pode gravar e reproduzir uma série inteira de valores.

Clique aqui para ver o exemplo

Async Subject É um Subject que emite o último valor após todos os valores serem enviados.

Apenas o último valor da execução observable é enviado para os observadores e somente quando a execução é concluida.

Clique aqui para ver o exemplo

Audit "Audita" ou "filtra" os valores enviados.

O Audit filtra os valores enviados, mantendo apenas os últimos n valores enviados.

Clique aqui para ver o exemplo

AuditTime "Audita" ou "filtra" os valores enviados.

Ignora os valores de origem por milissegundos de duração, em seguida, emite o valor mais recente da origem Observável e repete esse processo.

Clique aqui para ver o exemplo

Buffer Armazena os valores observáveis de origem até que o closureNotifier seja emitido.

O Buffer armazena os valores enviados e emite uma lista de valores quando o número de valores é atingido.
